# 手机点餐系统 🍽️

一个基于 **Spring Boot** 和 **微信小程序** 的手机点餐系统，支持用户登录、菜品浏览、下单、评论等功能。适用于食堂、餐馆等移动点餐场景。

---

## 🚀 技术栈

| 模块       | 技术           |
|------------|----------------|
| 后端       | Spring Boot, MySQL|
| 前端       | 微信小程序 (WXML + WXSS + JS) |
| 安全认证   | 自定义 Token 鉴权 |
| 数据库     | MySQL          |
| 项目构建   | Maven          |

---

## 🔧 功能模块

- ✅ 用户登录（支持手机和邮箱）
- ✅ 菜品浏览与分类查询
- ✅ 模糊搜索菜品
- ✅ 添加购物车 / 下单 / 修改订单
- ✅ 查看订单（按状态分类：准备中、已完成、已取消）
- ✅ 评论菜品，打分
- ✅ 个人中心
